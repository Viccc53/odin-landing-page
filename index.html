Genshin Impact is great

Genshin Impact is an action role-playing game developed and published by miHoYo. The game features an anime-style open-world environment and an action-based battle system using elemental magic and character-switching. The game is free-to-play and is monetized through gacha game mechanics through which players can obtain new characters and weapons.
